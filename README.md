# Swami Vivekanand College (MIDC, Latur) - Website

Frontend codebase for the official website of Swami Vivekanand College, featuring a modern UI, dark mode, and smooth animations.

Created by: **Chaughule Balaji**

## Tech Stack
React, Vite, Tailwind CSS v4, Framer Motion

## Setup Instructions

1. Make sure you have Node.js installed.
2. Open the project folder in your terminal.
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the dev server:
   ```bash
   npm run dev
   ```
